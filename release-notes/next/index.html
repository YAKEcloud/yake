Update provider-alicloud to 1.47.0

[gardener-extension-provider-alicloud]

âš ï¸ Breaking Changesâ€‹

  • [OPERATOR] With https://github.com/gardener/gardener-extension-provider-alicloud/pull/310 provider-alicloud migrated the volumesnapshot CRDs to a new dedicated ManagedResources. provider-alicloud does now remove the ignored CRDs. (gardener/gardener-extension-provider-alicloud#606, @ialidzhikov)
    • Before updating to this version of provider-alicloud, make sure that the migration of the volumesnapshot CRDs from the extension-controlplane-shoot to the extension-controlplane-shoot-crds ManagedResource completed. If the migration did not complete yet, GRM will interpret the removal of the CRDs as deletion and will delete the CRDs.
  • [OPERATOR] The .kubeAPIServer field in the component config has been removed since it's no longer needed anywhere. (gardener/gardener-extension-provider-alicloud#619, @rfranzke)
